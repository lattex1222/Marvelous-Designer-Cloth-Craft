## 🛠️ Getting Started: Your First 3D Garment  

### 🧵 Creating a Basic T-Shirt  
1. **Open a new project** and import a 3D avatar (OBJ/FBX supported).  
2. **Use the Pattern Tool** to draft rectangles for the front/back panels.  
3. **Stitch seams** with the *Pressing Tool* and simulate the fabric.  
4. **Adjust tension** and folds for realism using the *Pinning Tool*.  

### 🔥 Pro Tips for Beginners  
- **Keyboard Shortcuts**: Master `Ctrl+C` (copy) and `Ctrl+V` (paste) for patterns.  
- **Fabric Presets**: Start with "Cotton" or "Denim" for predictable results.  
- **UV Mapping**: Unwrap your garment early to avoid texture issues later.
